Контрольные вопросы и ответы по CSS Grid

1. Какими свойствами задаются количество и ширина столбцов в Grid‑контейнере?

Ответ: свойством grid-template-columns задают количество и ширину колонок; можно перечислить размеры через пробел (например, 100px 1fr 2fr) или использовать функции вроде repeat() для краткости

2. Какой метод можно использовать, чтобы равномерно распределить доступное пространство между столбцами или строками?

Ответ: использовать единицы доли пространства fr, например grid-template-columns: repeat(3, 1fr), чтобы три колонки получили равную ширину

3. Какие функции CSS Grid позволяют автоматически подстраивать количество колонок или строк под размер экрана?

Ответ: сочетание repeat() с ключевыми словами auto-fill или auto-fit и функцией minmax() (например, grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))) позволяет автоматически менять число колонок при изменении ширины экрана

4. Каким свойством можно задать промежутки между элементами Grid?

Ответ: свойство gap (или отдельно row-gap и column-gap) задаёт расстояния между строками и колонками

5. Как разместить элемент в нескольких строках или столбцах Grid?

Ответ: задать области или линии с помощью grid-column и grid-row, например grid-column: 1 / 3 (занять колонки с 1 по 2) или использовать именованные области через grid-template-areas

6. Найдете в сети красивое применение grid

Пример: адаптивные карточки галереи с равными колонками и автоматическим переносом (используют repeat(auto-fit, minmax(...))). Посмотрите статьи и примеры в руководствах по Grid для вдохновения.
