1 Структура базового HTML-документа и какие теги обязательно должны быть включены в него?

Базовый HTML-документ состоит из обязательных тегов, которые определяют структуру веб-страницы:

<!DOCTYPE html> - объявление типа документа
<html> - корневой элемент
<head> - метаинформация
<title> - заголовок страницы
<body> - содержимое страницы

Обязательные теги:

  • <!DOCTYPE html> - определяет версию HTML
  • <html lang="..."> - корневой элемент с указанием языка
  • <head> - содержит метаданные, заголовок, ссылки на стили
  • <meta charset="UTF-8"> - кодировка документа
  • <title> - заголовок страницы (отображается во вкладке браузера)
  • <body> - основное содержимое страницы
Пример минимальной структуры:
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Моя страница</title>
</head>
<body>
  <h1>Привет, мир!</h1>
</body>
</html>
2 Какой формат изображений нельзя использовать для размещения на HTML-странице?

Современные браузеры поддерживают различные форматы изображений, но есть исключения:

JPEG
Поддерживается
PNG
Поддерживается
GIF
Поддерживается
RAW
НЕ поддерживается
SVG
Поддерживается
WEBP
Поддерживается
RAW формат - это "сырой" формат данных с камеры, который требует специальной обработки и не поддерживается браузерами для непосредственного отображения на веб-страницах.

Правильный ответ: Формат RAW нельзя использовать для размещения на HTML-странице.

3 На какой версии языка программирования HTML Вы писали лабораторную работу №1?

Лабораторная работа №1 была выполнена на HTML5 - современной версии языка разметки.

HTML5 - это пятая и текущая основная версия HTML, которая включает:

  • Семантические теги (<header>, <footer>, <article> и др.)
  • Поддержка мультимедиа (<video>, <audio>)
  • Графические элементы (<canvas>, <svg>)
  • Улучшенная поддержка форм
  • API для сложных веб-приложений
<!DOCTYPE html> <-- Объявление HTML5

Объявление <!DOCTYPE html> в начале документа указывает браузеру, что страница написана на HTML5.

4 Какой тег в HTML используется для вставки разрыва строки?

Для вставки разрыва строки в HTML используется тег <br>.

Тег <br> - это одиночный тег (не требует закрывающего тега), который создает перенос строки.

Пример использования:

<p>Это первая строка<br>А это вторая строка</p>

Результат:

Это первая строка
А это вторая строка
Важно: Тег <br> следует использовать только для смысловых переносов строк (в стихах, адресах), а не для визуального форматирования. Для визуального оформления лучше использовать CSS.
5 Если Вы написали код страницы в текстовом редакторе и запустили полученный файл, смогут ли другие пользователи сети Интернет найти её, вставив в поисковую строку её адрес?

Нет, другие пользователи Интернета не смогут найти эту страницу, просто вставив адрес в поисковую строку.

Причина: Файл, открытый локально в браузере, имеет адрес вида file:///C:/path/to/your/file.html, который доступен только на вашем компьютере.

Чтобы сделать страницу доступной в Интернете, необходимо:

  1. Разместить файлы на веб-хостинге - специальном сервере, доступном из Интернета
  2. Получить доменное имя - уникальный адрес вашего сайта
  3. Загрузить файлы на сервер через FTP или панель управления хостингом

Локальный адрес: file:///C:/сайт/mypage.html (доступен только вам)

Интернет-адрес: https://mydomain.com/mypage.html (доступен всем)

Только после размещения на веб-сервере страница станет доступной для поисковых систем и других пользователей Интернета.
