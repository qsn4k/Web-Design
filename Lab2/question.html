1 Структура базового HTML-документа и обязательные теги
▼

Базовый HTML-документ состоит из следующих обязательных тегов:

<!DOCTYPE html> <html lang="ru"> <head> <meta charset="UTF-8"> <title>Заголовок страницы</title> </head> <body> Содержимое страницы </body> </html>

Обязательные элементы:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент
  • <head> - секция с метаинформацией
  • <meta charset="UTF-8"> - указание кодировки
  • <title> - заголовок страницы
  • <body> - основное содержимое
2 Какой формат изображений нельзя использовать на HTML-странице?
▼

Среди перечисленных форматов RAW нельзя использовать для размещения на HTML-странице.

JPEG
Поддерживается
PNG
Поддерживается
GIF
Поддерживается
RAW
Не поддерживается
SVG
Поддерживается
WEBP
Поддерживается
Примечание: RAW - это "сырой" формат данных с камеры, который требует специальной обработки и не поддерживается браузерами для непосредственного отображения.
3 На какой версии HTML написана лабораторная работа №1?
▼

Лабораторная работа №1 выполнена на HTML5.

HTML5 - это современная версия языка разметки, которая включает:

  • Семантические теги (<header>, <footer>, <article>)
  • Поддержка мультимедиа (<video>, <audio>)
  • Улучшенная поддержка форм
  • API для веб-приложений

Объявление <!DOCTYPE html> в начале документа указывает на использование HTML5.

4 Какой тег используется для вставки разрыва строки?
▼

Для вставки разрыва строки используется тег <br>.

Пример использования:

<p>Первая строка<br>Вторая строка</p>

Результат:

Первая строка
Вторая строка
Рекомендация: Тег <br> следует использовать для смысловых переносов (стихи, адреса), а не для визуального оформления.
5 Доступность HTML-страницы в Интернете после создания
▼

Нет, другие пользователи Интернета не смогут найти страницу, созданную локально.

Причина: Локальный файл имеет адрес вида file:///C:/path/to/file.html, который доступен только на вашем компьютере.

Чтобы сделать страницу доступной в Интернете:

  1. Разместить файлы на веб-хостинге
  2. Зарегистрировать доменное имя
  3. Загрузить файлы на сервер

Локальный адрес: file:///C:/сайт/page.html

Интернет-адрес: https://example.com/page.html
